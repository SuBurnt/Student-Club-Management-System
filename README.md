# Student-Club-Management-System

## Описание

Веб-приложение для управления студенческими клубами, создания мероприятий и обсуждений.

## Функционал

* Регистрация и авторизация студентов
* Создание и управление клубами
* Присоединение к клубам
* Создание мероприятий
* Регистрация на мероприятия
* Обсуждения и комментарии

## Технологии

* Python 3.10+
* Django 4.2+
* SQLite
* Bootstrap 5

## Установка

1. Клонируйте репозиторий через терминал:
git clone [https://github.com/SuBurnt/Student-Club-Management-System.git](https://github.com/SuBurnt/Student-Club-Management-System.git)
2. Установите необходимые пакеты  
pip install -r requirements.txt  
3. Перейдите к нужной папке  
cd student_club_management  
4. Запустите сервер:  
python manage.py runserver  
5. Скопируйте адрес и вставьте в браузер:  
Starting development server at [***http://127.0.0.1:8000/***](http://127.0.0.1:8000/)
